# webpack demos

webapck3 + react

## 环境依赖

* Node.js v8+
* npm v5+

## 安装

```bash
$ git clone https://github.com/lovefishs/webpack-demos.git
$ cd webpack-demos
$ npm install
$ npm run demo-01
$ npm run demo-02
$ ...
```
